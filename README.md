## 설치하고 실행하는 방법

Replace the contents of your `_config.yml` file with [this](https://github.com/raviriley/agency-jekyll-theme/blob/master/_config.yml).

And then execute:

    $ bundle

Or, install it yourself as:

    $ gem install jekyll-agency
    
실행하려면

```bash
$ jekyll serve
```
